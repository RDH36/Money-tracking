# Rapport Feedback Testeurs - Mitsitsy

**Date:** Février 2025
**Source:** Testers Community
**Statut Global:** App stable, aucun crash ni bug critique

---

## Priorité 1 - Critique (Légal/Compliance)

### Privacy Policy & Conditions d'Utilisation
- [x] Rédiger une Privacy Policy complète
- [x] Rédiger les Conditions Générales d'Utilisation
- [x] Ajouter liens accessibles dans les Settings
- [x] Ajouter liens sur la page d'onboarding

**Pourquoi:** Obligatoire pour le Play Store et conformité RGPD.

---

## Priorité 2 - Haute (UX Majeure)

### Dark Mode
- [x] Toggle dans les Settings (Light/Dark/System)
- [x] Thèmes de couleurs personnalisables (optionnel)
- [x] Mode automatique selon l'heure (optionnel)

**Pourquoi:** Confort visuel nocturne, économie batterie OLED, feature très demandée.

### Onboarding / Tutoriel
- [x] Walkthrough interactif au premier lancement (4 slides animés avec swipe)
- [x] Option "Skip" avec accès ultérieur dans Settings ("Revoir le tutoriel")
- [x] Animations fluides (icône pulsante, features staggerées, dots animés)
- [ ] Tooltips contextuels sur les fonctionnalités clés (optionnel)

**Pourquoi:** Réduit la frustration des nouveaux utilisateurs, améliore la rétention.

---

## Priorité 3 - Moyenne (Expansion)

### Support Multi-Langue
- [ ] Sélecteur de langue dans Settings
- [ ] Traduction FR/EN minimum
- [ ] Détection automatique selon device (optionnel)
- [ ] Localisation des messages d'erreur et notifications

**Pourquoi:** Élargit l'audience internationale.
